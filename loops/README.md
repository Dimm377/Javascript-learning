# 05 - Loops

Semua jenis perulangan dalam JavaScript.

## Files
- `for-loop.js` - For loop dasar
- `while-loop.js` - While loop
- `do-while.js` - Do-while loop
- `for-in.js` - For-in loop (iterasi object keys)
- `for-of.js` - For-of loop (iterasi array values)
- `infinite-loops.js` - Contoh infinite loop (hati-hati!)
- `break-continue.js` - Break dan continue statements

## Run
```bash
node for-loop.js
node while-loop.js
# etc.
```
